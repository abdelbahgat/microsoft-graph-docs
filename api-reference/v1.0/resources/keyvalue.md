# keyValue resource type

Namespace: microsoft.graph

Represents a key-value pair.

## Properties

| Property	   | Type	|Description|
|:---------------|:--------|:----------|
|key|string| Key for the key-value pair. |
|value|string| Value for the key-value pair.|
