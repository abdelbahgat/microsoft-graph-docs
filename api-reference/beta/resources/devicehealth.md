# deviceHealth resource type

Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

## Properties
| Property     | Type        | Description |
|:-------------|:------------|:------------|
|lastConnectionTime|DateTimeOffset|The last time the device was connected.|
